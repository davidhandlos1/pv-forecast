# PV Forecast

Wieviel Strom erzeugt meine Solaranlage in den nächsten Tagen? PV Forecast zeigt eine übersichtliche 7-Tage-Prognose für Photovoltaikanlagen — direkt im Browser, ohne Registrierung.

**[Zur App](https://davidhandlos1.github.io/pv-forecast/)**

## Funktionen

- **7-Tage-Ertragsprognose** — stündliche Leistungsvorhersage für die eigene PV-Anlage
- **Bis zu 3 Dachflächen** — einzeln oder kombiniert anzeigen
- **Wetter im Überblick** — Temperatur, Bewölkung, UV-Index und Sonnenscheindauer
- **Aktuelle Strompreise** vom Großhandelsmarkt
- **Automatische Aktualisierung** alle 30 Minuten
- **Keine Anmeldung nötig** — einfach aufrufen und nutzen
- **Funktioniert überall** — Handy, Tablet und Desktop

## Eigene Anlage einrichten

In den Einstellungen lassen sich Standort, Dachneigung, Ausrichtung und installierte Leistung (kWp) für bis zu drei Dachflächen hinterlegen. Die Konfiguration wird als URL gespeichert und kann einfach als Lesezeichen abgelegt oder geteilt werden.

## Datenquellen

- Wetter & Einstrahlung: [Open-Meteo](https://open-meteo.com/)
- Strompreise: [aWATTar](https://www.awattar.de/)

Alle Daten sind kostenlos und ohne API-Key verfügbar.
